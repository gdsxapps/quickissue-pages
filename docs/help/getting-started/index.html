QuickIssue Setup

Setup

First, create a token on GitHub.

Go to the GitHub authentication page.

Please authenticate.

Enter the code you received via email here.

Authentication using mobile app

Please enter this number in the GitHub mobile app.

Once authenticated, you will be redirected to this screen.

Enter the token name and expiration period.

  1. Enter token name
    Names that have been used before cannot be reused
  2. Expiration period
    Setting a longer period is more convenient

Press the Generate Token button at the bottom.

The token is displayed on this screen.

Copy it. (Please press the copy button)

Back To App

Return to the app and paste the token in step 2 Paste your token.

After pasting, save it.

Proceed to step 3

Press Add destination to select a destination.

A list of available repositories will be displayed.

Select the repository you want to register as a destination.

In the free version, you can select only one.

You are now ready to use the app.
