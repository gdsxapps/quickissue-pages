# GitHub Token Setup

This page explains how to generate and configure a GitHub Personal Access Token (PAT) for use with QuickIssue.

## Steps

1. Go to [GitHub Settings > Developer settings > Personal access tokens](https://github.com/settings/tokens)
2. Click "Generate new token"
3. Select the required scopes (repo access)
4. Copy the generated token
5. Paste the token into QuickIssue
