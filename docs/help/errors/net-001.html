What happened?

This error occurs when QuickIssue cannot establish a connection to GitHub. Common causes include:

  • No internet connection
  • WiFi or network is disconnected
  • Firewall or VPN blocking the connection
  • GitHub is experiencing an outage

How to fix it

  1. Check your internet connection
    • Try opening a website in your browser
    • Check your WiFi or Ethernet connection
  2. Check GitHub status
    • Visit GitHub Status to see if there's an outage
  3. Disable VPN or proxy temporarily if you're using one
  4. Check firewall settings to ensure QuickIssue isn't blocked
  5. Restart the app and try again
