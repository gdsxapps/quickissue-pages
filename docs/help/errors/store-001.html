What happened?

This error occurs when QuickIssue cannot save your Personal Access Token to the system's secure storage (Keychain on macOS).

How to fix it

  1. Check Keychain Access
    • Open "Keychain Access" app on your Mac
    • Look for any QuickIssue-related entries
    • Delete them if corrupted
  2. Restart QuickIssue and try saving the token again
  3. Restart your Mac if the issue persists
  4. Check disk permissions
    • Go to System Preferences ‚Üí Security & Privacy ‚Üí Privacy
    • Ensure QuickIssue has necessary permissions
