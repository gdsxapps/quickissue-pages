Error: STORE-004

⚠️ Failed to add repository - QuickIssue couldn't save the repository to your destinations.

What happened?

This error occurs when QuickIssue cannot add a repository to your saved destinations list.

How to fix it

  1. Check your subscription
    • Free version allows only 1 repository
    • Remove existing repository before adding a new one, or upgrade to Pro
  2. Verify repository access
    • Ensure your token has access to the repository
    • Check that the repository still exists on GitHub
  3. Restart QuickIssue and try again
