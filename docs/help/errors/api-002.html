Error: API-002

⚠️ Failed to update issue - QuickIssue couldn't update the GitHub issue.

What happened?

This error occurs when an issue update request fails. Possible causes:

  • The issue was deleted or closed
  • You no longer have permission to edit the issue
  • Network interruption during the request
  • GitHub API is experiencing issues

How to fix it

  1. Check the issue on GitHub
    • Verify the issue still exists
    • Check if you have write access to the repository
  2. Verify your token permissions
    • Ensure your token has the repo scope
  3. Check GitHub status at githubstatus.com
  4. Try again after a few moments
