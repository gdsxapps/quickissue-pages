# GitHub トークンの設定

このページでは、QuickIssue で使用する GitHub Personal Access Token (PAT) の生成と設定方法を説明します。

## 手順

1. [GitHub Settings > Developer settings > Personal access tokens](https://github.com/settings/tokens) にアクセス
2. 「Generate new token」をクリック
3. 必要なスコープを選択（リポジトリアクセス）
4. 生成されたトークンをコピー
5. QuickIssue にトークンを貼り付け
